# Creative Link Hover Effects

A collection of modern and subtle hover effects for links or menu items, featuring smooth animations and clean design.

## Features

- Multiple unique hover effects
- Responsive design
- Smooth animations using CSS
- Easy to customize and implement
- Lightweight and fast

## Demo

Open `index.html` in your browser to see the hover effects in action.

## Effects Included

1. **Effect 1**: Simple underline animation
2. **Effect 2**: Text reveal with data attribute
3. **Effect 3**: (Add more effects as needed)

## Getting Started

1. Clone the repository:

   ```bash
   git clone [your-repository-url].git
   ```

2. Open `index.html` in your preferred web browser.

## Usage

To use these effects in your project:

1. Copy the CSS from `src/css/style.css` to your project's stylesheet.
2. Use the appropriate HTML structure as shown in `index.html`.
3. Add the corresponding class to your navigation elements.

## Customization

You can easily customize the colors, timing, and other properties in the CSS file to match your project's design system.

## Browser Support

The effects work on all modern browsers including:

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
